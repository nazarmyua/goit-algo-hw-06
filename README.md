## DFS vs BFS (in this graph)

- DFS explores the metro network by going as deep as possible along one line before backtracking. In this case, it follows long linear chains of stations and reaches terminal stations early.

- BFS explores the network level by level, visiting all nearby stations first. Here, it expands evenly through transfer hubs, giving a more balanced view of the network structure.

- Key difference: DFS emphasizes depth (long paths), while BFS emphasizes proximity (shortest number of steps from the start).
